# OverView
This App is ToDoList.
It is possible to manage to separate 'Hobbies' & 'Schedule'.

# Help
1 -> Add Hobbies
2 -> Add Schedule
3 -> Delete
4 -> Show
Other numbers -> Close

# Description
Using the Sort Algorithm by date.
The list to create is saved in input_s.txt and input_h.txt.

# Note
Please enter the name of the ToDo in Roman characters.
